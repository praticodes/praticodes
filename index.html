PROJECTS

I'm currently focusing on projects that dive into artificial intelligence, machine learning, natural language processing, data analysis, and optimization.

In this section, you can see a selection of the projects I've recently built and deployed.

Compel-O-Meter
  • Uses the recursive tree data structure, natural language processing, and artificial intelligence to analyze inputted text for persuasiveness and dangerous sentiment. We (three peers and I) hope that, by flagging propaganda and hate speech (specifically in tweets), this project will act as a tool for critical thinking in a digital age.
  • Launch
LitLoom
  • Uses linear programming/optimization techniques to provide an optimal book list for reading club's, based on diverse factors include genres, ratings, book awards, and more. This application was inspired by what I learned in an Applications of Linear Programming course I took in the Winter 2023 session.
  • Launch
SentiGram
  • This project aims to find trends in the sentiments expressed in tweets over the course of any one week, using R for statistical analysis and Python's NLTK for sentiment analysis. It then aims to present the data clearly through visualizations, to be made available on a full-stack web application.
  • Coming Soon
